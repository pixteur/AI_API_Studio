AI API Studio

AI image studio for Gemini, Fal, and BytePlus workflows

1.2 beta - Open Source

About the Project

AI API Studio is a local web application that lets you generate AI images directly from your browser, using multiple providers and image models from one workspace, including Gemini Nano Banana, Fal-hosted Nano Banana and Seedream, and BytePlus Seedream 4.5.

It began with Sergio Valsecchi's initial open-source work and creative direction. Pixteur has since taken that foundation further, expanding it into the broader production app it is now with more providers, editing tools, workflow depth, and a more studio-focused experience.

The app still runs entirely on your machine: no cloud subscription, no forced account layer, and no usage limits imposed by the app itself. Assets, prompts, favorites, and Elements stay local, while image requests only go to the providers you explicitly configure, such as Gemini, Fal, or BytePlus. If you have strong ideas for where AI API Studio should go next, please reach out to Pixteur directly.

Installation

The only prerequisite is Python 3.11 (or 3.12). No other manual setup is needed - the app installs its own dependencies automatically on first launch.

Win

Windows

Install Python 3.11 from python.org/downloads and tick "Add python.exe to PATH" during setup.

Then double-click start.bat inside the app folder.

Or from Command Prompt: python nbs.py

Mac

macOS

Install Python 3.11 from python.org/downloads or via Homebrew: brew install python@3.11.

Then double-click start.sh in Finder, or run ./start.sh in Terminal.

Missing packages such as Flask, Pillow, and Requests are installed automatically on first launch. No manual pip install or virtual environment is required. The app opens at http://localhost:5000.

How to Use

1

Add your provider API keys

Open Settings and add the keys you want to use: Gemini from aistudio.google.com/apikey, plus optional Fal and BytePlus keys for the extra Nano Banana and Seedream providers.

2

Select a model and provider

Use Nano Banana, Nano Banana Pro, or Nano Banana 2 on Gemini or Fal, and switch to Seedream 4.5 or Seedream 5 Lite when you want the Fal or BytePlus routes.

3

Build your input

Write your prompt, add reference images from the Generator, Loved, Elements, or drag and drop, then tune seed and safety controls based on the provider you picked.

4

Generate, queue, and refine

Click Generate, then keep editing and queue more jobs while earlier ones run in the background. Save favorites to Loved, reuse prompts, compare upscales, and export from the result cards.

5

Organize your asset library

Build out Elements with talents, locations, and props, use Loved as a reusable reference bank, and switch into Edit mode when you want to upscale selected images with SeedVR.
